# API testing

### Setup Virtualenv

    $ virtualenv ./venv
    $ ./venv/bin/pip install -r requirements.txt

### Test/Play:

    $ project path .\APItesting\reflektion\ref_api
    $ project path > pytest test.py --html=result/APItesting_report.html

### logs and reports
    $ ./API_test.log
    $ result/API_test.html
